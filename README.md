## 🚀 My current experimental projects

[promener](https://github.com/jycamier/promener) : An "OpenAPI for metrics" — contract-first approach to Prometheus metrics for automatic code and documentation generation.

[gobed](https://github.com/jycamier/gobed) : Stop and start cloud services and machines on a schedule — like a sleep timer for your infrastructure.

[wrapper](https://github.com/jycamier/wrapper) : Wrap any CLI command with profile-based configuration management.

[retrotro](https://github.com/jycamier/retrotro) : Real-time collaborative retrospective tool for agile teams.

[meshcap](https://github.com/jycamier/meshcap) : Kubernetes-native HTTP traffic capture for service meshes.

[wafflex](https://github.com/jycamier/wafflex) : A CLI to test WAF rules by replaying real HTTP traffic through Coraza.
